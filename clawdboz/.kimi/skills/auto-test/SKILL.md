# Auto Test Skill - 自动化测试工具

## 描述
自动化构建、安装和测试 clawdboz whl 包的完整流程。

## 功能

1. **自动打包**: 在项目目录执行 `python -m build`
2. **创建测试环境**: 在 `../test` 目录创建 uv 虚拟环境
3. **安装测试**: 安装生成的 whl 包
4. **自测验证**: 运行测试用例验证功能

## 使用方式

```bash
# 运行完整测试流程
python auto_test.py

# 或使用 skill 调用
python -m clawdboz.auto_test
```

## 测试流程

```
1. 清理旧的构建产物
2. 构建 whl 包
3. 创建 ../test 目录
4. 使用 uv 创建虚拟环境
5. 安装 whl 包
6. 生成并运行测试用例
7. 输出测试结果
```

## 测试用例包括

- 包导入测试
- Bot 初始化测试
- 配置文件生成测试
- 日志目录创建测试
- 命令行工具测试

## 输出

测试结果会输出到控制台，并在 `../test/test_results.log` 中保存详细日志。
